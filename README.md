# Median_coffee_spending_report

### Скрипт для обработки csv-файла.

Скрипт читает файлы с данными о тратах студентов на кофе и формирует отчёт, который включает в себя список студентов и медиану трат (по колонке coffee spent), отчёт сортируется по убыванию трат. 

Название файлов (может быть несколько) передается в виде параметров --files. 

Название отчета передается в виде параметра --report (в нашем случае это median-coffee). 

Отчёт формируется один по всем переданным файлам.

В архитектуру заложена возможность добавления новых отчётов через параметр --report. В дальнейшем можно будет добавлять отчёты с другой логикой расчетов, поэтому их добавление должно быть удобным.

Код покрыт тестами написанными на pytest с покрытием более 80% в соответствии с pytest-cov.

Код соответствует общепринятым стандартам написания проектов на python и общепринятому стилю.

> [!IMPORTANT]
Чтобы отображать табличные данные используем модуль tabulate. Он не входит в стандартную библиотеку Python, поэтому tabulate нужно установить: pip install tabulate.
>
> Установите pytest и pytest-cov: pip install pytest pytest-cov

#### Использованы следующие модули:

**argparse** — встроенная библиотека Python для создания удобных интерфейсов командной строки.

**csv** —  модуль для чтения и записи табличных данных (Comma-Separated Values).

**statistics** —  модуль, предназначенный для вычисления базовых статистических показателей числовых данных, таких как среднее арифметическое, медиана, мода, дисперсия и стандартное отклонение.

____________

#### Назначение файлов:

math.csv, programming.csv, physics.csv — содержат предназначенные для обработки файлы с данными о тратах.

main.py - содержит скрипт, обрабатывающий файлы c данными о тратах.

test_ main.py — содержит скрипт тестов.

screenshot_main.JPG — скриншот запуска основного скрипта в main.py

screenshot_test.JPG — скриншот запуска скрипта тестирования.

Запуск кода: 

python main.py --files math.csv programming.csv physics.csv --report median-coffee

____________

### Тестирование.

Запуск тестов:  
pytest --cov=main
